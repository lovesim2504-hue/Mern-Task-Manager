# Mern-Task-Manager
A full-stack Task Management Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). This application helps users organize tasks, track progress, manage deadlines, and improve productivity through an intuitive and responsive interface.

## 📌 Features

### User Features
- User Authentication (Register/Login)
- JWT-based Authorization
- Create, Read, Update, and Delete Tasks
- Task Status Management (To Do, In Progress, Completed)
- Task Priority Levels
- Due Date Management
- Search and Filter Tasks
- Responsive User Interface
- Profile Management

### Advanced Features
- Role-Based Access Control (RBAC)
- File Upload Support (Cloudinary)
- Email Verification
- Password Reset via Email
- Dashboard Analytics
- Task Sorting and Pagination
- Secure API Endpoints
- Real-Time Notifications (Optional)
